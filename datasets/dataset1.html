PubChem Dataset

50 Million Chemical Compounds(181.48GB)

Dataset Overview

The dataset is provided in SDF (Structure-Data File) format, which preserves the 3D spatial coordinates of each atom in the molecules — a very powerful feature for 3D chemical analysis. There are also numerous other features included, which will be detailed in this README, allowing users to explore the data fully. I intentionally kept the data in its raw SDF format and did not convert or clean it to CSV, so you can experience the full data cleaning and preprocessing process yourself, and see all available information, not just what I consider important. Dataset Size: Each SDF file contains approximately 500,000 compounds, making it a very large-scale dataset suitable for deep learning and chemical modeling.

Inspiration

This dataset opens the door to multiple exciting possibilities: Compound Behavior Analysis: Study how different compounds interact or behave based on their structural features. Chemical Translator Development: Build models that can interpret chemical structures or convert between representations. 3D Molecular Structure Prediction (Challenging but Brilliant!): Predict the spatial 3D structure of compounds based on available features — a tough task with high potential for innovation.

Data Quality && Data Source

This dataset is sourced from PubChem and is not my own data. I have simply transferred it here for easier access and experimentation. The dataset is extremely rich, both in terms of the number of samples and the number of features, making it a valuable resource for various chemical and computational analyses.
